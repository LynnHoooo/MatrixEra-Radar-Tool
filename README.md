# R60ABD1 矩阵纪元雷达测试工具 V1.0

这是一个专为矩阵纪元设计的现代化跨平台上位机测试软件。该工具使用 Python 结合 PyQt6 构建，用于帮助硬件及软件工程师快速测试与验证雷达的人员活动捕捉、心率检测以及呼吸监测等生命体征特征。

## ✨ 核心功能亮点

1. **暗黑界面 (Dark Mode) 现代化 UI：**
   - 直观的卡片化数据显示（有人状态、心率 BPM、呼吸 次/分、目标距离），状态字带有显眼的色彩分类。
   - 自适应窗口，实时渲染串口收发记录及 Hex 原文。
2. **多线程安全串口通信：**
   - 解析不阻塞 UI。严格的 Windows COM 口防占用的独有安全通信机制。
3. **SY 官方全协议解析：**
   - 包含 `0x80` 人体存在、`0x81` 呼吸检测、`0x84` 睡眠及 `0x85` 心率数据的精准分包截取，以及 checksum 校验计算抛弃坏包。
4. **一键式 Excel/CSV 数据录制：**
   - 自动生成带有时间戳的毫米波雷达解析记录与原始 Hex 包数据。
   - 文件采用 `utf-8-sig`，兼容 Windows Excel 直接打开免乱码。
   - 流式写入防丢失设计，意外断电、拔线也不会损失此前录制数据。

## 📥 安装及环境要求

需要提前安装 Python 3.9+，并在终端安装依赖：

```bash
pip install pyserial PyQt6
```

运行程序：

```bash
python radar_tester.py
```

## 📡 协议说明简述

本上位机依据 R60ABD1_数据手册_V1.1 及串口通信规范实现，标准通信层配置为 `115200 bps, 8数据位, 1停止位, 无校验`。

**通用帧结构格式要求：**

`[帧头 53 59] + [控制字] + [标识字] + [长度H] + [长度L] + [数据] + [校验和] + [帧尾 54 43]`

上位机目前已实装解析的关键控制字与标识字如下：

| 功能类 (Control) | 标识符 (Ident) | 功能说明 | 数据示例域说明 |
| :--- | :--- | :--- | :--- |
| **0x80 (人体存在)** | `0x01` | 有人/无人状态 | `0x00`: 无人, `0x01`: 有人 |
| | `0x02` | 活跃等级 | `0+`: 静止有人 / 活跃有人 |
| | `0x03` | 体动参数 | 数值(0-100)表示运动幅度 |
| | `0x04` | 探测距离 | 结合高低位计算 `cm` 单位距离 |
| **0x81 (呼吸检测)** | `0x01` | 呼吸判别状态 | 正常 / 呼吸过高 / 呼吸过低等 |
| | `0x02` | **呼吸数值** | 转化为 `次/分钟 (rpm)` |
| | `0x05` | 呼吸波形数据 | 正弦波基准值128上下浮动 |
| **0x84 (睡眠监测)** | `0x01` | 入床/离床 | `0x01`: 入床, `0x00`: 离床 |
| | `0x02` | 睡眠状态判定 | 深睡 / 浅睡 / 清醒 |
| **0x85 (心率监测)** | `0x02` | **心率数值** | 转化为 `BPM` 单位 |
| | `0x05` | 心率波形数据 | 实时心动动态波形 |
| **0x07 (雷达范围)** | `0x07` | 目标越界预警 | `0x01`: 范围内, `0x00`: 范围外 |

> 注意：上位机中已为您内置了快捷发送指控（如开启/关闭各个模式的报文下发）。您只需在左侧面板选择雷达所连接的 COM 口并连接，即可点击按钮操作雷达，面板将会自动识别回包并刷新仪表盘。
